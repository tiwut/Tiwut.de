Ubuntu Desktop Installation Guide

Introduction to Ubuntu

Ubuntu is one of the most popular open-source Linux distributions, developed by Canonical. It's known for its ease of use, strong community support, and wide hardware compatibility, making it an excellent choice for beginners and experienced users alike.

Ubuntu Desktop typically uses the GNOME desktop environment, but official "flavors" offer other environments like KDE Plasma (Kubuntu), XFCE (Xubuntu), MATE (Ubuntu MATE), Budgie (Ubuntu Budgie), and LXQt (Lubuntu).

This guide focuses on installing the standard Ubuntu Desktop (with GNOME). The process for other flavors is very similar, primarily differing in the look and feel of the live session and installed desktop.

Prerequisites & Crucial: Backup Your Data!

CRITICAL: BACKUP YOUR DATA! Installing any new operating system, especially if repartitioning, carries the risk of data loss. Before you begin, back up all important documents, photos, applications, and personal files to an external hard drive, USB flash drive, or cloud storage.

You will need the following:

  • A Computer:
    • Processor: 2 GHz dual-core processor or better (64-bit).
    • RAM: 4 GB system memory (8 GB+ recommended for smoother experience).
    • Storage: 25 GB of free hard drive space (more recommended for applications and data). SSD highly recommended for better performance.
    • Display: 1024x768 screen resolution.
    • Internet access is helpful, but not strictly required for installation.
  • Ubuntu ISO File: You'll download this in the next step.
  • USB Flash Drive: Minimum 4GB capacity (8GB+ recommended). This drive will be erased.
  • (Optional) Another Working Computer: To download the ISO and create the USB installation media.

1. Download Ubuntu ISO & Verify

  1. Go to the official Ubuntu Desktop download page.
  2. You'll typically see options for the latest LTS (Long Term Support) release and the newest interim release.
    • LTS Releases: Supported for 5 years. Recommended for stability and most users.
    • Interim Releases: Released every 6 months, supported for 9 months. Offer newer features but may be less stable.
    For most users, the LTS version is the best choice.
  3. Download the 64-bit .iso file.

  4. Verify the ISO Download (Recommended):

    • On the download page or a linked page (often named SHA256SUMS), find the SHA256 checksum for your downloaded ISO.
    • Use a tool to calculate the SHA256 checksum of your local ISO file:
      • Linux/macOS Terminal: sha256sum /path/to/ubuntu.iso
      • Windows PowerShell: Get-FileHash -Algorithm SHA256 /path/to/ubuntu.iso
      • Windows CMD (CertUtil): CertUtil -hashfile /path/to/ubuntu.iso SHA256
    • Compare the calculated hash with the official one. They must match exactly. If not, re-download.

2. Create Bootable Installation Media (USB Drive)

The USB drive will be completely erased during this process. Back up any data on it first.

Popular tools for creating a bootable USB:

  • BalenaEtcher (Recommended for ease): Available for Windows, macOS, and Linux. Select the Ubuntu ISO, select your USB drive, and click "Flash!".
  • Rufus (Windows): Select your USB drive. Click "SELECT" and choose the Ubuntu ISO. Rufus usually detects the correct settings (GPT partition scheme, UEFI target). Click "START". If asked about ISOHybrid image, choose "Write in DD Image mode".
  • Ubuntu's Startup Disk Creator (if you're already on Ubuntu): Search for "Startup Disk Creator" in your applications. Select the ISO and USB drive.
  • Ventoy (Advanced - Multi-boot): Install Ventoy to the USB drive once, then copy multiple ISOs (Ubuntu, Windows, etc.) directly to it.

3. BIOS/UEFI Configuration

  1. Insert the bootable Ubuntu USB drive into the computer.
  2. Access BIOS/UEFI Setup: Restart or turn on the computer. Press the key for BIOS/UEFI setup (Common: Del, F2, F10, F12, Esc).
  3. Set Boot Order: Make the USB drive the first boot device (e.g., "USB HDD," "Removable Devices," or "UEFI: [USB Drive Name]").
  4. UEFI vs. Legacy/CSM Mode:
    • UEFI Mode is strongly recommended. Ensure "Boot Mode" is set to "UEFI".
    • Disable "CSM" (Compatibility Support Module) or "Legacy Boot".
  5. Secure Boot:
    • Ubuntu supports Secure Boot and usually works fine with it enabled.
    • If you encounter issues booting the USB or after installation (especially with third-party drivers like NVIDIA), you might need to temporarily disable Secure Boot as a troubleshooting step.
  6. SATA Mode: Ensure SATA mode is set to AHCI.
  7. Disable Fast Boot/Quick Boot (in BIOS/UEFI and/or Windows power settings if dual-booting).
  8. Save and Exit.

4. Boot from Installation Media & Live Session

  1. The computer will restart and should boot from the USB drive.
  2. You'll see the GRUB boot menu for Ubuntu. Options usually include:
    • Try or Install Ubuntu: (Often the default) This will lead to further options.
    • Ubuntu (safe graphics)
    • OEM install (for manufacturers)
    • Test memory
    Select "Try or Install Ubuntu" or let it auto-boot.
  3. After a loading screen, you'll be presented with a welcome screen giving two main options:
    • Try Ubuntu: Boots into a live desktop environment running from the USB. This is highly recommended to:
      • Test hardware compatibility (Wi-Fi, Bluetooth, sound, graphics, touchpad).
      • Get a feel for the GNOME desktop.
      • Use tools like GParted for manual partitioning before installation.
    • Install Ubuntu: Starts the installation process directly.
    Click "Try Ubuntu" first. Once on the live desktop, you can launch the installer by clicking the "Install Ubuntu" icon.

5. Installer: Initial Steps

Once you launch the installer (Ubiquity):

  1. Language: Select your preferred language. Click "Continue".
  2. Keyboard Layout: Choose your keyboard layout. You can "Detect Keyboard Layout" or select manually and test it. Click "Continue".
  3. Wireless (Network Connection):
    • If using Ethernet, it should connect automatically.
    • Select your Wi-Fi network and enter the password to connect.
    • Alternatively, you can choose "I don't want to connect to a Wi-Fi network right now" and connect later. An internet connection is recommended. Click "Continue".
  4. Updates and other software:
    • What apps would you like to install to start with?
      • Normal installation: Includes web browser, utilities, office software, games, and media players. Recommended for most users.
      • Minimal installation: Includes web browser and basic utilities.
    • Other options:
      • Download updates while installing Ubuntu: Saves time later. Recommended if connected to the internet.
      • Install third-party software for graphics and Wi-Fi hardware and additional media formats: Installs proprietary drivers (e.g., for NVIDIA graphics, some Wi-Fi cards) and media codecs. Highly recommended for better hardware support and media playback.
      • Click "Continue".

6. Installation Type (Disk Partitioning)

This is the most critical step. Incorrect choices can lead to data loss. Ensure backups are complete!

The installer will present several options:

  1. Erase disk and install Ubuntu:
    • Wipes the entire selected hard drive and installs Ubuntu as the only OS.
    • Advanced features (optional):
      • "Use LVM with the new Ubuntu installation": Logical Volume Management for flexible partitioning.
      • "Encrypt the new Ubuntu installation for security" (Full Disk Encryption): Encrypts the entire Ubuntu system. You'll set an encryption passphrase.
    • If you choose this, select the target drive carefully. Click "Install Now". A summary of changes will be shown; click "Continue" if correct.
  2. Install Ubuntu alongside [Existing OS e.g., Windows Boot Manager]:
    • This option appears if another OS is detected. It allows you to dual-boot.
    • The installer will show a slider to resize the existing OS partition to make space for Ubuntu.
    • It's often safer to shrink the Windows partition from within Windows Disk Management *before* starting the Ubuntu installation to avoid potential issues.
    • Click "Install Now". A summary of changes will be shown; click "Continue".
  3. Something else (Manual Partitioning):
    • Provides full control. Necessary for specific layouts or advanced dual-boot setups.
    • You'll see a list of drives and existing partitions. You can create, delete, resize, and assign mount points.
    • For UEFI systems (most common), typical manual partitions:
      1. EFI System Partition (ESP):
        • Use: Select the existing ESP if dual-booting with Windows (usually FAT32, ~100-500MB). Do NOT format it if sharing. Just ensure it's selected as "EFI System Partition" in the "Use as" dropdown.
        • If no ESP exists (e.g., clean disk), create one:
          • Size: 256MB to 512MB (e.g., 300MB).
          • Type: Primary.
          • Location: Beginning of the disk space.
          • Use as: EFI System Partition.
      2. Root Partition (/):
        • Size: Minimum 25GB (30-50GB+ recommended, or much larger if no separate /home).
        • Type: Primary or Logical.
        • Use as: Ext4 journaling file system.
        • Mount point: /
      3. (Highly Recommended) Home Partition (/home):
        • Size: Use remaining desired space for your personal files.
        • Type: Primary or Logical.
        • Use as: Ext4 journaling file system.
        • Mount point: /home
        • Benefit: Allows OS reinstallation without losing personal data.
      4. Swap Area:
        • Ubuntu now typically uses a swap file by default, so a swap partition is often not strictly necessary for fresh installs unless you have specific needs (like hibernation with full disk encryption, or very low RAM).
        • If creating a swap partition manually:
          • Size: Traditionally, RAM size (for hibernation) or a smaller amount (e.g., 2-8GB) if hibernation isn't a priority and you have ample RAM.
          • Use as: swap area.
    • After setting up partitions, select the device for boot loader installation. For UEFI systems, this is usually the drive containing the ESP (e.g., /dev/sda, not /dev/sda1 - though the installer often handles this correctly by targeting the ESP).
    • Click "Install Now". A summary of changes will be shown; click "Continue".

7. Where are you? (Location & Timezone)

Select your geographical location on the map, or type your city. This sets the system timezone. Click "Continue".

8. Who are you? (User Account Setup)

  1. Your name: Your real name (optional).
  2. Your computer's name (hostname): e.g., my-ubuntu-pc.
  3. Pick a username: Your login name (lowercase, no spaces, e.g., john).
  4. Choose a password: A strong password for your account.
  5. Confirm your password.
  6. Options:
    • "Log in automatically" (Not recommended for security).
    • "Require my password to log in" (Recommended).
  7. (Optional) "Use Active Directory" - for corporate environments.
    8. Click "Continue".

9. Installation Process

Ubuntu will now install. A slideshow will display features of Ubuntu while files are copied and the system is configured. This can take 10-30 minutes or more, depending on your system speed and options selected.

10. Installation Complete & Restart

  1. When the installation is finished, you'll see a message "Installation Complete".
  2. Click "Restart Now".
  3. You will be prompted to "Please remove the installation medium, then press ENTER". Remove the USB drive and press Enter.
  4. The computer will reboot into your new Ubuntu system. If you dual-boot, you should see the GRUB boot menu allowing you to choose between Ubuntu and your other OS.

11. Post-Installation Steps

  1. Log in with the username and password you created.
  2. Welcome Screen/Tour: You might see a welcome screen offering to set up Livepatch, connect online accounts, or take a tour.

  3. Update Your System (Crucial):

    Open a Terminal (Ctrl+Alt+T) and run:
    sudo apt update
    sudo apt full-upgrade -y
    Alternatively, the "Software Updater" application might prompt you.
  4. Install Drivers (if needed):
    • Search for "Additional Drivers" or "Software & Updates" and go to the "Additional Drivers" tab.
    • If you have NVIDIA graphics or certain Wi-Fi cards, proprietary drivers might be listed here. Select the recommended one and click "Apply Changes". A reboot may be required.
  5. Install Software:
    • Use the "Ubuntu Software" application (Snap Store) to find and install graphical applications.
    • Use the terminal for more control: sudo apt install <packagename>.
    • Consider installing ubuntu-restricted-extras for common media codecs (if not selected during install): sudo apt install ubuntu-restricted-extras (you'll need to accept EULAs).
  6. Customize Your Desktop:
    • Right-click desktop > "Change Background" or "Display Settings".
    • Install GNOME Extensions via the "Extensions" app (may need gnome-shell-extensions and gnome-tweaks installed: sudo apt install gnome-shell-extensions gnome-tweaks) and browser integration for extensions.gnome.org.
  7. Set up Backups: Ubuntu includes "Backups" (Deja Dup). Configure it to back up your important files to an external drive or cloud storage.
  8. Enable Firewall (UFW): sudo ufw enable
    sudo ufw status

Troubleshooting Tips

  • System Doesn't Boot / GRUB Issues: Check BIOS/UEFI boot order. Secure Boot (try disabling if issues persist). Use a live USB and "Boot Repair" tool (sudo apt install boot-repair && boot-repair).
  • No Wi-Fi / Bluetooth: Ensure third-party drivers were enabled during install. Check "Additional Drivers". Search online for "[Your Ubuntu Version] [Your Wi-Fi Card Model] driver".
  • Graphics Issues: Install correct proprietary drivers (NVIDIA/AMD) via "Additional Drivers".
  • Software Installation Problems: Check internet. Run sudo apt update.
