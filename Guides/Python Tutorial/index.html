A Comprehensive Beginner's Guide to Python Programming

Introduction to Python

Welcome to the world of Python! Python is a high-level, interpreted programming language known for its readability, simplicity, and versatility. Created by Guido van Rossum and first released in 1991, Python's design philosophy emphasizes code readability with its notable use of significant indentation.

Why learn Python?

  • Easy to Learn: Python has a simple syntax similar to the English language, making it an excellent choice for beginners.
  • Versatile: It's used in web development, data science, machine learning, artificial intelligence, scripting, game development, and much more.
  • Large Community & Libraries: Python has a massive and active community, which means plenty of resources, support, and a vast collection of third-party libraries (pre-written code) to help you accomplish tasks quickly.
  • Cross-Platform: Python code can run on Windows, macOS, Linux, and other operating systems without modification.
  • Interpreted: Python code is executed line by line, which makes debugging easier.

Setting Up Your Environment

Before you can start coding in Python, you need to install it on your computer.

1. Installing Python

  1. Visit the official Python website: python.org/downloads/
  2. Download the latest version of Python 3 for your operating system (Windows, macOS, Linux).
  3. Run the installer.

    Windows Users: Make sure to check the box that says "Add Python to PATH" during installation. This makes it easier to run Python from the command line.

  4. To verify the installation, open your terminal (Command Prompt on Windows, Terminal on macOS/Linux) and type: 
    python --version
    Or, depending on your installation: 
    python3 --version
    You should see the Python version number printed.

2. Choosing a Code Editor / IDE

While you can write Python code in a simple text editor, using a dedicated code editor or an Integrated Development Environment (IDE) will make your life much easier. They offer features like syntax highlighting, code completion, and debugging tools.

Popular choices include:
  • Visual Studio Code (VS Code): Free, powerful, and highly customizable with excellent Python support through extensions. Download VS Code
  • PyCharm: A professional IDE specifically for Python. It has a free Community Edition. Download PyCharm
  • Sublime Text: A lightweight and fast editor, popular for its speed and flexibility. Download Sublime Text
  • IDLE: Python's built-in basic IDE. It's installed automatically with Python and is good for very simple scripts or quick tests.

For this guide, we'll assume you're using a plain text file with a .py extension (e.g., my_script.py) and running it from the terminal.

Your First Python Program: "Hello, World!"

It's a tradition in programming to start with a "Hello, World!" program. This simple program will print the text "Hello, World!" to your screen.

  1. Open your chosen code editor (or a simple text editor like Notepad).
  2. Type the following line of code: 
    print("Hello, World!")
  3. Save the file as hello.py (make sure the extension is .py).
  4. Open your terminal or command prompt.
  5. Navigate to the directory where you saved hello.py. For example, if you saved it in a folder named "PythonProjects" on your Desktop: 
    cd Desktop/PythonProjects
  6. Run the script by typing: 
    python hello.py
    (Or python3 hello.py if needed)

You should see the output:

Hello, World!

Explanation:

  • print() is a built-in Python function. Functions are reusable blocks of code that perform a specific task.
  • The text inside the parentheses ("Hello, World!") is called an argument that we pass to the print() function.
  • "Hello, World!" is a string, which is a sequence of characters enclosed in quotes (either single ' ' or double " ").

Variables and Data Types

Variables are containers for storing data values. In Python, you don't need to explicitly declare the type of a variable; the type is inferred from the value assigned to it.

Creating Variables

You create a variable by assigning a value to it using the equals sign (=).

# Variable assignment
name = "Alice"       # String
age = 30             # Integer
height = 5.9         # Float (decimal number)
is_student = False   # Boolean (True or False)

print(name)
print(age)
print(height)
print(is_student)

Output:

Alice
30
5.9
False

Common Data Types

Python has several built-in data types. Here are the most common ones:
  • Text Type: str (string) - sequence of characters. 
    my_string = "Hello Python"
another_string = 'Python is fun!'
  • Numeric Types:
    • int (integer) - whole numbers, positive or negative, without decimals. 
      count = 100
negative_number = -5
    • float (floating-point number) - numbers with a decimal point. 
      price = 19.99
pi_value = 3.14159
    • complex (complex numbers) - written with a "j" as the imaginary part. 
      z = 3 + 5j
  • Sequence Types:
    • list - ordered, mutable (changeable) collection. Allows duplicate members. 
      my_list = ["apple", "banana", "cherry"]
    • tuple - ordered, immutable (unchangeable) collection. Allows duplicate members. 
      my_tuple = ("apple", "banana", "cherry")
  • Mapping Type:
    • dict (dictionary) - unordered, mutable collection of key-value pairs. Keys must be unique. 
      my_dict = {"name": "Bob", "age": 25}
  • Set Types:
    • set - unordered, mutable collection. No duplicate members. 
      my_set = {"apple", "banana", "cherry"}
  • Boolean Type:
    • bool - represents one of two values: True or False. 
      is_active = True
has_permission = False
  • None Type:
    • NoneType - represents the absence of a value. Its only value is None. 
      result = None

You can find the type of any variable using the type() function:

x = 5
print(type(x))  # Output: <class 'int'>

y = "Hello"
print(type(y))  # Output: <class 'str'>

Variable Naming Rules

  • Must start with a letter (a-z, A-Z) or an underscore (_).
  • Cannot start with a number.
  • Can only contain alpha-numeric characters and underscores (A-z, 0-9, and _).
  • Are case-sensitive (age, Age, and AGE are three different variables).
  • Avoid using Python keywords (like if, else, while, def, etc.) as variable names.

Use descriptive variable names to make your code more readable (e.g., user_name instead of un).

Operators

Operators are special symbols in Python that carry out arithmetic or logical computation. The value that the operator operates on is called the operand.

1. Arithmetic Operators

Used to perform mathematical operations:

  • + : Addition (x + y)
  • - : Subtraction (x - y)
  • * : Multiplication (x * y)
  • / : Division (results in a float, x / y)
  • % : Modulus (remainder of division, x % y)
  • ** : Exponentiation (x ** y, same as xy)
  • // : Floor division (division that results in whole number, discarding remainder, x // y)
a = 10
b = 3

print(a + b)   # Output: 13
print(a - b)   # Output: 7
print(a * b)   # Output: 30
print(a / b)   # Output: 3.333...
print(a % b)   # Output: 1 (10 divided by 3 is 3 with a remainder of 1)
print(a ** b)  # Output: 1000 (10 to the power of 3)
print(a // b)  # Output: 3 (discards the .333...)

2. Comparison Operators

Used to compare two values. They return either True or False.

  • == : Equal to (x == y)
  • != : Not equal to (x != y)
  • > : Greater than (x > y)
  • < : Less than (x < y)
  • >= : Greater than or equal to (x >= y)
  • <= : Less than or equal to (x <= y)
x = 5
y = 10

print(x == y)  # Output: False
print(x != y)  # Output: True
print(x > y)   # Output: False
print(x < y)   # Output: True

3. Logical Operators

Used to combine conditional statements. They also return True or False.

  • and : Returns True if both statements are true.
  • or : Returns True if one of the statements is true.
  • not : Reverses the result, returns False if the result is true, and vice-versa.
age = 25
has_license = True

# Example 1: and
if age > 18 and has_license:
    print("Can drive") # Output: Can drive
else:
    print("Cannot drive")

# Example 2: or
is_weekend = False
is_holiday = True
if is_weekend or is_holiday:
    print("Day off!") # Output: Day off!
else:
    print("Work day")

# Example 3: not
is_raining = False
if not is_raining:
    print("It's not raining, good for a walk!") # Output: It's not raining...

4. Assignment Operators

Used to assign values to variables. We've already seen =. There are also compound assignment operators:

  • += : x += 3 is same as x = x + 3
  • -= : x -= 3 is same as x = x - 3
  • *= : x *= 3 is same as x = x * 3
  • And so on for /=, %=, //=, **=.
count = 5
count += 2  # count is now 7
print(count)

count *= 3  # count is now 21 (7 * 3)
print(count)

Input and Output

Output using print()

We've already used the print() function to display output. It can print strings, numbers, variables, and expressions.

name = "Charlie"
age = 22
print("My name is", name, "and I am", age, "years old.")
# Output: My name is Charlie and I am 22 years old.

# Using f-strings (formatted string literals - recommended for Python 3.6+)
print(f"My name is {name} and I am {age} years old.")
# Output: My name is Charlie and I am 22 years old.

print(5 + 3) # Output: 8

F-strings provide a concise and convenient way to embed expressions inside string literals for formatting.

Input using input()

The input() function allows you to get input from the user. It always returns the user's input as a string, even if they type numbers.

user_name = input("Enter your name: ")
print(f"Hello, {user_name}!")

# Getting numerical input
user_age_str = input("Enter your age: ")
# You need to convert the string to an integer if you want to do math
user_age_int = int(user_age_str) 

birth_year = 2024 - user_age_int
print(f"You were likely born in {birth_year}.")

If you run the code above, it will pause and wait for you to type something after "Enter your name: " and "Enter your age: ".

Always remember that input() returns a string. If you need a number, you must convert it using int() for integers or float() for decimal numbers. If the user enters text that cannot be converted (e.g., "abc" when expecting a number), your program will raise an error (ValueError).

Control Flow

Control flow statements determine the order in which statements are executed in a program.

1. Conditional Statements (if, elif, else)

These statements allow you to execute certain blocks of code only if a specific condition is true.

Indentation is crucial in Python. It defines the blocks of code associated with control flow statements.

temperature = 25

if temperature > 30:
    print("It's a hot day!")
elif temperature > 20:  # 'elif' is short for 'else if'
    print("It's a pleasant day.")
elif temperature < 10:
    print("It's a cold day!")
else:
    print("It's cool.")

# Output for temperature = 25: It's a pleasant day.

Key points:

  • The condition is followed by a colon (:).
  • The code block to be executed if the condition is true is indented (usually 4 spaces).
  • elif allows you to check multiple conditions.
  • else is optional and executes if none of the preceding if or elif conditions are true.

2. Looping Statements

Loops are used to execute a block of code repeatedly.

a) for loop

A for loop is used for iterating over a sequence (like a list, tuple, dictionary, set, or string) or other iterable objects.

# Looping through a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
# Output:
# apple
# banana
# cherry

# Looping through a string
for letter in "Python":
    print(letter)
# Output:
# P
# y
# t
# h
# o
# n

# Looping with range()
# range(5) generates numbers from 0 up to (but not including) 5
for i in range(5):
    print(i)
# Output:
# 0
# 1
# 2
# 3
# 4

# range(start, stop, step)
for i in range(2, 10, 2): # Start at 2, up to 10, increment by 2
    print(i)
# Output:
# 2
# 4
# 6
# 8

b) while loop

A while loop executes a block of code as long as a specified condition is true.

count = 0
while count < 5:
    print(count)
    count += 1  # Important: update the counter, otherwise it's an infinite loop!
# Output:
# 0
# 1
# 2
# 3
# 4

print("Loop finished")

Be careful with while loops! If the condition never becomes false, you'll create an infinite loop, and your program will hang. Always ensure there's a way for the condition to eventually become false.

3. Loop Control Statements

  • break: Exits the current loop prematurely. 
    for i in range(10):
    if i == 5:
        break  # Stop the loop when i is 5
    print(i)
# Output: 0, 1, 2, 3, 4
  • continue: Skips the rest of the code inside the loop for the current iteration and proceeds to the next iteration. 
    for i in range(5):
    if i == 2:
        continue  # Skip printing 2
    print(i)
# Output: 0, 1, 3, 4
  • pass: The pass statement is a null operation; nothing happens when it executes. It's used as a placeholder where syntactically some code is required, but you haven't written it yet. 
    def my_empty_function():
    pass # TODO: Implement this later

if x > 10:
    pass # Placeholder for future code

Functions

A function is a block of organized, reusable code that is used to perform a single, related action. Functions provide better modularity for your application and a high degree of code reusing.

Defining a Function

You define a function using the def keyword, followed by the function name and parentheses (). 
def greet():
    print("Hello from a function!")

# Calling the function
greet() # Output: Hello from a function!

Function Arguments (Parameters)

You can pass data, known as parameters or arguments, into a function.

def greet_user(name): # 'name' is a parameter
    print(f"Hello, {name}!")

greet_user("Alice")  # "Alice" is an argument. Output: Hello, Alice!
greet_user("Bob")    # Output: Hello, Bob!

Return Values

Functions can return a value to the caller using the return statement.

def add_numbers(x, y):
    sum_result = x + y
    return sum_result

result = add_numbers(5, 3)
print(result)  # Output: 8

another_result = add_numbers(10, 20)
print(another_result) # Output: 30

If a function doesn't have a return statement, or has a return statement without an expression, it implicitly returns None.

Default Parameter Values

You can specify a default value for a parameter. If an argument for that parameter is not provided when the function is called, the default value is used.

def greet_with_message(name, message="Welcome"):
    print(f"{message}, {name}!")

greet_with_message("Eve")                # Output: Welcome, Eve!
greet_with_message("Frank", "Good morning") # Output: Good morning, Frank!

Keyword Arguments

You can also call functions using keyword arguments of the form kwarg=value. This allows you to pass arguments out of order.

def describe_pet(animal_type, pet_name):
    print(f"I have a {animal_type} named {pet_name}.")

describe_pet(animal_type="hamster", pet_name="Harry") # Output: I have a hamster named Harry.
describe_pet(pet_name="Lucy", animal_type="dog")     # Output: I have a dog named Lucy.

Docstrings (Documentation Strings)

It's good practice to add a documentation string (docstring) as the first statement in a function's body. It explains what the function does.

def multiply(a, b):
    """
    This function multiplies two numbers and returns the result.
    
    Args:
        a (int or float): The first number.
        b (int or float): The second number.
        
    Returns:
        int or float: The product of a and b.
    """
    return a * b

print(multiply(4, 5)) # Output: 20
help(multiply)        # Prints the docstring

Data Structures

Python offers several built-in data structures to store collections of data efficiently.

1. Lists

Lists are ordered, mutable (changeable), and allow duplicate elements. They are defined using square brackets [].

# Creating a list
my_list = [1, "hello", 3.14, True, "hello"]
print(my_list) # Output: [1, 'hello', 3.14, True, 'hello']

# Accessing elements (zero-indexed)
print(my_list[0])  # Output: 1
print(my_list[1])  # Output: hello

# Modifying elements
my_list[1] = "world"
print(my_list) # Output: [1, 'world', 3.14, True, 'hello']

# List methods
my_list.append("new_item")    # Adds an item to the end
print(my_list) # Output: [1, 'world', 3.14, True, 'hello', 'new_item']

my_list.insert(2, "Python")   # Inserts "Python" at index 2
print(my_list) # Output: [1, 'world', 'Python', 3.14, True, 'hello', 'new_item']

my_list.remove("world")       # Removes the first occurrence of "world"
print(my_list)

popped_item = my_list.pop()   # Removes and returns the last item
print(popped_item) # Output: new_item
print(my_list)

print(len(my_list)) # Gets the length of the list

# Slicing
print(my_list[1:3]) # Elements from index 1 up to (not including) 3

2. Tuples

Tuples are ordered, immutable (unchangeable), and allow duplicate elements. They are defined using parentheses ().

# Creating a tuple
my_tuple = (1, "apple", 2.5, "apple")
print(my_tuple) # Output: (1, 'apple', 2.5, 'apple')

# Accessing elements (same as lists)
print(my_tuple[0]) # Output: 1

# Tuples are immutable - this would cause an error:
# my_tuple[0] = 5 # TypeError: 'tuple' object does not support item assignment

# Use cases: When you have a collection of items that should not change,
# like coordinates (x, y) or RGB color values.
coordinates = (10, 20)

3. Dictionaries

Dictionaries are unordered (in Python versions before 3.7; ordered from 3.7+), mutable, and store data in key-value pairs. Keys must be unique and immutable (e.g., strings, numbers, or tuples). Defined using curly braces {}.

# Creating a dictionary
person = {
    "name": "Alice",
    "age": 30,
    "city": "New York"
}
print(person)

# Accessing values using keys
print(person["name"])  # Output: Alice
print(person.get("age")) # Output: 30 (get() is safer, returns None if key not found)

# Modifying values
person["age"] = 31
print(person)

# Adding new key-value pairs
person["occupation"] = "Engineer"
print(person)

# Removing items
person.pop("city")
print(person)

# Iterating through a dictionary
for key in person:
    print(key, ":", person[key])

for key, value in person.items(): # More Pythonic way
    print(f"{key}: {value}")

4. Sets

Sets are unordered, mutable, and do not allow duplicate elements. Defined using curly braces {} or the set() function.

# Creating a set
my_set = {1, 2, 3, "apple", 2, "banana"} # Duplicates are automatically removed
print(my_set) # Output might be in different order, e.g., {1, 2, 3, 'banana', 'apple'}

# Adding elements
my_set.add(4)
my_set.add("apple") # No change, "apple" is already there
print(my_set)

# Removing elements
my_set.remove(2)      # Raises KeyError if element not found
my_set.discard("orange") # Does nothing if element not found (safer)
print(my_set)

# Set operations
set1 = {1, 2, 3}
set2 = {3, 4, 5}
print(set1.union(set2))          # Output: {1, 2, 3, 4, 5}
print(set1.intersection(set2))   # Output: {3}
print(set1.difference(set2))     # Output: {1, 2}

Modules and Packages

Python's power comes from its extensive standard library and the vast ecosystem of third-party packages.

Modules

A module is simply a Python file (.py) containing Python definitions and statements. You can use these definitions (functions, classes, variables) in other Python files.

To use a module, you import it.

Example: Using the math module

import math # Import the entire module

print(math.pi)    # Accessing pi constant. Output: 3.141592653589793
print(math.sqrt(16)) # Using the square root function. Output: 4.0

# You can also import specific items from a module
from math import pi, sqrt

print(pi)         # Now you can use pi directly. Output: 3.14159...
print(sqrt(25))   # Output: 5.0

# Importing with an alias
import math as m
print(m.factorial(5)) # Output: 120 (5*4*3*2*1)

Creating Your Own Module

Let's say you have a file named my_module.py with the following content:

my_module.py:

# This is my_module.py
def say_hello(name):
    print(f"Hello, {name} from my_module!")

MY_CONSTANT = "This is a constant from my_module."

Now, in another Python file (e.g., main_program.py) in the same directory, you can import and use it:

main_program.py:

import my_module

my_module.say_hello("Python Learner")
# Output: Hello, Python Learner from my_module!

print(my_module.MY_CONSTANT)
# Output: This is a constant from my_module.

Packages

A package is a collection of related modules. It's essentially a directory containing modules and a special __init__.py file (which can be empty) that tells Python that this directory should be treated as a package.

Example directory structure:

my_package/
├── __init__.py
├── module1.py
└── module2.py

You can then import modules from the package:

import my_package.module1
from my_package import module2

my_package.module1.some_function()
module2.another_function()

Pip and Third-Party Packages

pip is the package installer for Python. You can use pip to install packages from the Python Package Index (PyPI).

Common pip commands (run in your terminal/command prompt):

  • Install a package: pip install package_name (e.g., pip install requests)
  • Uninstall a package: pip uninstall package_name
  • List installed packages: pip list
  • Show information about an installed package: pip show package_name

When working on larger projects, it's highly recommended to use virtual environments to manage dependencies for different projects separately. Tools like venv (built-in) or conda are used for this.

Error Handling (Exceptions)

Errors that occur during program execution are called exceptions. Python uses try...except blocks to handle exceptions gracefully, preventing your program from crashing.

try:
    # Code that might raise an exception
    numerator = 10
    denominator = 0 # This will cause a ZeroDivisionError
    result = numerator / denominator
    print(result)
except ZeroDivisionError:
    # Code to run if a ZeroDivisionError occurs
    print("Error: Cannot divide by zero!")
except ValueError:
    # Code to run if a ValueError occurs
    print("Error: Invalid value entered.")
except Exception as e: # Catch any other general exception
    print(f"An unexpected error occurred: {e}")
else:
    # Code to run if no exceptions were raised in the try block
    print("Division successful.")
finally:
    # Code that will always run, regardless of whether an exception occurred or not
    print("Execution finished.")

# Output for the above:
# Error: Cannot divide by zero!
# Execution finished.

Key parts:

  • try: The block of code to be attempted.
  • except: If an error occurs in the try block, Python looks for a matching except block. You can specify the type of exception to catch.
  • else (optional): Executed if the try block completes without raising an exception.
  • finally (optional): Always executed, whether an exception occurred or not. Useful for cleanup actions (e.g., closing files).

You can also raise exceptions deliberately using the raise keyword:

def check_age(age):
    if age < 0:
        raise ValueError("Age cannot be negative.")
    elif age < 18:
        print("Minor.")
    else:
        print("Adult.")

try:
    check_age(-5)
except ValueError as e:
    print(f"Error: {e}") # Output: Error: Age cannot be negative.

File I/O (Input/Output)

Python provides built-in functions for creating, reading, and writing files.

Opening and Closing Files

The open() function is used to open a file. It returns a file object.

Syntax: file_object = open("filename.txt", "mode")

Common modes:

  • 'r': Read (default mode). Raises an error if the file does not exist.
  • 'w': Write. Creates a new file if it does not exist or truncates (empties) the file if it exists.
  • 'a': Append. Creates a new file if it does not exist. Appends to the end of the file if it exists.
  • 'r+': Read and Write.
  • 'b': Binary mode (e.g., 'rb' or 'wb' for image files).

It's crucial to close a file using file_object.close() after you're done with it to free up system resources and ensure data is written.

Using with statement (Recommended)

The with statement provides a cleaner way to work with files. It automatically closes the file even if errors occur.

Writing to a File

# Writing to a file (will create or overwrite example.txt)
try:
    with open("example.txt", "w") as file:
        file.write("Hello, Python files!\n")
        file.write("This is a second line.\n")
    print("File written successfully.")
except IOError:
    print("Error: Could not write to file.")

# Appending to a file
try:
    with open("example.txt", "a") as file:
        file.write("Appending a new line.\n")
    print("Content appended successfully.")
except IOError:
    print("Error: Could not append to file.")

Reading from a File

try:
    with open("example.txt", "r") as file:
        # Read the entire content
        content = file.read()
        print("--- Full Content ---")
        print(content)

        # Or, read line by line (file cursor is now at the end, so re-open or seek)
    
    with open("example.txt", "r") as file:
        print("--- Line by Line ---")
        for line in file: # Iterating over a file object reads it line by line
            print(line.strip()) # .strip() removes leading/trailing whitespace like \n
            
    with open("example.txt", "r") as file:
        print("--- Readlines ---")
        lines_list = file.readlines() # Reads all lines into a list
        for line_from_list in lines_list:
            print(line_from_list.strip())

except FileNotFoundError:
    print("Error: The file was not found.")
except IOError:
    print("Error: Could not read from file.")

Brief Introduction to Object-Oriented Programming (OOP) (Optional)

Python is an object-oriented programming language. OOP is a programming paradigm based on the concept of "objects", which can contain data (attributes) and code (methods).

This is a vast topic. This section provides only a very basic overview.

Classes and Objects

  • Class: A blueprint for creating objects. It defines a set of attributes and methods that the created objects will have.
  • Object: An instance of a class.
# Define a class named 'Dog'
class Dog:
    # Class attribute (shared by all instances)
    species = "Canis familiaris"

    # Initializer / Constructor method
    def __init__(self, name, age):
        # Instance attributes (specific to each instance)
        self.name = name
        self.age = age
        print(f"{self.name} instance created!")

    # Instance method
    def bark(self):
        return f"{self.name} says Woof!"

    # Another instance method
    def get_details(self):
        return f"{self.name} is {self.age} years old."

# Create objects (instances) of the Dog class
dog1 = Dog("Buddy", 3)  # Calls __init__ automatically
dog2 = Dog("Lucy", 5)   # Calls __init__ automatically

# Access instance attributes
print(f"{dog1.name} is a {Dog.species}.") # Output: Buddy is a Canis familiaris.
print(f"{dog2.name} is {dog2.age} years old.") # Output: Lucy is 5 years old.

# Call instance methods
print(dog1.bark())     # Output: Buddy says Woof!
print(dog2.get_details()) # Output: Lucy is 5 years old.

Key OOP Concepts:

  • Encapsulation: Bundling data (attributes) and methods that operate on the data within a single unit (class).
  • Inheritance: Creating new classes (child classes) that inherit properties and methods from existing classes (parent classes). 
    class Poodle(Dog): # Poodle inherits from Dog
    def __init__(self, name, age, haircut_style):
        super().__init__(name, age) # Call parent's __init__
        self.haircut_style = haircut_style

    def bark(self): # Overriding the parent's bark method
        return f"{self.name} says Yap yap!"

my_poodle = Poodle("Fifi", 2, "fancy")
print(my_poodle.get_details()) # Inherited method. Output: Fifi is 2 years old.
print(my_poodle.bark())        # Overridden method. Output: Fifi says Yap yap!
print(my_poodle.haircut_style) # Output: fancy
  • Polymorphism: The ability of objects of different classes to respond to the same method call in different ways (e.g., the bark() method above).
