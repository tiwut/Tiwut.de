Linux Mint Installation Guide

Introduction to Linux Mint

Linux Mint is a community-driven Linux distribution based on Ubuntu (with some editions historically based on Debian). It's renowned for its user-friendliness, "out-of-the-box" readiness with multimedia codecs, and its traditional desktop experience. Mint aims to be a modern, elegant, and comfortable operating system that is both powerful and easy to use.

Linux Mint offers several desktop environment editions:

  • Cinnamon: Mint's flagship desktop, modern, feature-rich, and highly customizable with a traditional layout.
  • MATE: A fork of GNOME 2, offering a classic, stable, and resource-friendly desktop.
  • XFCE: A lightweight, fast, and stable desktop environment, excellent for older hardware or users preferring simplicity.

This guide focuses on installing Linux Mint (any desktop edition). The installation process is largely identical across editions, mainly differing in the live session's appearance and the final installed desktop.

Prerequisites & Crucial: Backup Your Data!

CRITICAL: BACKUP YOUR DATA! Installing any new operating system carries the risk of data loss. Before you begin, back up all important files and personal data to an external drive or cloud storage.

You will need:

  • A Computer:
    • Processor: 64-bit processor (32-bit versions are no longer actively developed for new releases).
    • RAM: 2 GB RAM (4 GB+ recommended for a comfortable experience).
    • Storage: 20 GB of disk space (100 GB recommended for applications and data). An SSD is highly recommended.
    • Display: 1024Ã—768 resolution.
    • Internet access is helpful for downloading updates and multimedia codecs during installation.
  • Linux Mint ISO File: Downloaded from the official website.
  • USB Flash Drive: Minimum 4GB capacity (8GB+ recommended). This drive will be erased.
  • (Optional) Another Working Computer: To download the ISO and create the USB installation media.

1. Download Linux Mint ISO & Verify

  1. Go to the official Linux Mint download page.
  2. Choose your preferred desktop edition (Cinnamon, MATE, or XFCE).
  3. Select a download mirror close to your location for faster speeds. Download the 64-bit .iso file.

  4. Verify the ISO Download (Essential!):

    Linux Mint emphasizes ISO verification.
    1. On the download page (or a linked integrity/authenticity page), download the sha256sum.txt and sha256sum.txt.gpg files corresponding to your ISO.
    2. Verify the SHA256 sum:
      • Open a terminal in the directory where you downloaded the files.
      • Calculate the SHA256 sum of your ISO: sha256sum your-linuxmint.iso
      • Compare this output with the sum listed in the sha256sum.txt file for your specific ISO. They must match.
    3. Verify the authenticity of sha256sum.txt (Advanced but recommended): This step ensures the checksum file itself is genuine.
      • Import the Linux Mint signing key: gpg --keyserver hkp://keyserver.ubuntu.com:80 --recv-key "27DE B156 44C6 B3CF 3BD7 D291 300F 846B A25B AE09" (The key ID might change for newer releases; check Mint's official guide for the current key.)
      • Verify the signature: gpg --verify sha256sum.txt.gpg sha256sum.txt
      • A "Good signature" message indicates the checksum file is authentic.
    4. If verification fails at any step, do not use the ISO. Re-download from a different mirror.

2. Create Bootable Installation Media (USB Drive)

The USB drive will be completely erased. Back up its data first.

Recommended tools:

  • Linux Mint's USB Image Writer (if on Linux Mint/Ubuntu): Search for "USB Image Writer" in your menu. Select the ISO and USB drive.
  • BalenaEtcher (Windows, macOS, Linux): Select the Mint ISO, select your USB drive, click "Flash!".
  • Rufus (Windows): Select your USB drive. Click "SELECT" and choose the Mint ISO. Rufus should auto-configure. If asked about ISOHybrid, choose "Write in DD Image mode".
  • Ventoy (Advanced - Multi-boot): Install Ventoy to the USB once, then copy multiple ISOs directly to it.

3. BIOS/UEFI Configuration

  1. Insert the bootable Linux Mint USB drive.
  2. Access BIOS/UEFI Setup: Restart/turn on the computer. Press the key for BIOS/UEFI setup (Common: Del, F2, F10, F12, Esc).
  3. Set Boot Order: Make the USB drive the first boot device (e.g., "USB HDD," "UEFI: [USB Drive Name]").
  4. UEFI vs. Legacy/CSM Mode:
    • UEFI Mode is strongly recommended. Ensure "Boot Mode" is "UEFI".
    • Disable "CSM" (Compatibility Support Module) or "Legacy Boot".
  5. Secure Boot:
    • Linux Mint generally works with Secure Boot enabled.
    • If you encounter issues booting or with third-party drivers (like NVIDIA) after installation, you might need to disable Secure Boot.
  6. SATA Mode: Set to AHCI.
  7. Disable Fast Boot/Quick Boot.
  8. Save and Exit.

4. Boot from Installation Media & Live Session

  1. The computer will restart and boot from the USB drive.
  2. You'll see the Linux Mint GRUB boot menu. Options include:
    • Start Linux Mint [Edition Name]: (Often the default) Boots into the live session.
    • Start Linux Mint (compatibility mode)
    • OEM install
    • Check the integrity of the medium
    Select "Start Linux Mint" or let it auto-boot.
  3. The live desktop environment will load. This allows you to try Linux Mint without installing it.
    • Test hardware compatibility (Wi-Fi, sound, graphics, etc.).
    • Explore the desktop environment.
  4. On the live desktop, double-click the "Install Linux Mint" icon to start the installation.

5. Installer: Initial Steps

The installer (Ubiquity, same as Ubuntu) will guide you:

  1. Language: Select your preferred language. Click "Continue".
  2. Keyboard Layout: Choose your keyboard layout. You can test it. Click "Continue".
  3. Multimedia Codecs:
    • Linux Mint offers a checkbox: "Install multimedia codecs".
    • Checking this box is highly recommended as it installs common proprietary codecs for MP3, video playback (H.264), etc., providing a better out-of-the-box multimedia experience.
    Click "Continue". An internet connection is helpful here.

6. Installation Type (Disk Partitioning)

This is a critical step. Incorrect choices can lead to data loss. Ensure backups are complete!

The installer presents several options:

  1. Erase disk and install Linux Mint:
    • Wipes the entire selected hard drive and installs Mint as the only OS.
    • Advanced features (optional under this option):
      • "Use LVM with the new Linux Mint installation".
      • "Encrypt the new Linux Mint installation for security" (Full Disk Encryption).
    • Select the target drive carefully. Click "Install Now". Confirm changes by clicking "Continue".
  2. Install Linux Mint alongside [Existing OS]:
    • Appears if another OS (like Windows) is detected, allowing dual-boot.
    • A slider lets you resize the existing OS partition to make space for Mint.
    • Consider shrinking the Windows partition from Windows Disk Management *before* starting the Mint installation for safety.
    • Click "Install Now". Confirm changes by clicking "Continue".
  3. Something else (Manual Partitioning):
    • Full control for specific layouts or advanced dual-boot.
    • Typical manual partitions for UEFI systems:
      1. EFI System Partition (ESP):
        • Use existing ESP: If dual-booting with Windows, select its ESP (FAT32, ~100-500MB). Choose "EFI System Partition" as "Use as". Do NOT format if sharing.
        • Create new ESP: If clean disk: Size 256-512MB, Type Primary, Location beginning, Use as EFI System Partition.
      2. Root Partition (/):
        • Size: Min 20GB (30-50GB+ recommended, or larger if no separate /home).
        • Type: Primary/Logical. Use as: Ext4 journaling file system. Mount point: /.
      3. (Highly Recommended) Home Partition (/home):
        • Size: Remaining desired space. Type: Primary/Logical. Use as: Ext4. Mount point: /home.
      4. Swap Area:
        • Linux Mint, like Ubuntu, typically uses a swap file by default, so a swap partition isn't always necessary unless you have specific needs (e.g., hibernation with encryption, very low RAM).
        • If creating manually: Size based on RAM (e.g., RAM size or 2-8GB). Use as: swap area.
    • Device for boot loader installation: For UEFI, this is usually the drive with the ESP (e.g., /dev/sda).
    • Click "Install Now". Confirm changes by clicking "Continue".

7. Where are you? (Location & Timezone)

Select your geographical location on the map or type your city to set the system timezone. Click "Continue".

8. Who are you? (User Account Setup)

  1. Your name: Your real name (optional).
  2. Your computer's name (hostname): e.g., mint-desktop.
  3. Pick a username: Your login name (e.g., emma, lowercase, no spaces).
  4. Choose a password: A strong password.
  5. Confirm your password.
  6. Options:
    • "Log in automatically" (Not recommended for security).
    • "Require my password to log in" (Recommended).
    • (Optional) "Encrypt my home folder": Provides an extra layer of security for your personal files. You won't be able to access them if you forget your login password.
    7. Click "Continue".

9. Installation Process

Linux Mint will now install. A slideshow highlights Mint's features. This may take 10-30 minutes or more.

10. Installation Complete & Restart

  1. A message "Installation Complete" will appear.
  2. Click "Restart Now".
  3. You'll be prompted: "Please remove the installation medium, then press ENTER". Remove the USB drive and press Enter.
  4. The computer reboots into your new Linux Mint system. The GRUB boot menu may appear if dual-booting.

11. Post-Installation Steps

  1. Log in with your created username and password.
  2. Welcome Screen: Linux Mint's Welcome Screen is very helpful. It guides you through:
    • First Steps: Desktop colors, panel layout, system snapshots, Driver Manager, Update Manager, System Settings, Software Manager, Firewall.
    • Documentation & Help: Links to official guides and support.
    It's highly recommended to go through these sections.
  3. System Snapshots (Timeshift): The Welcome Screen will strongly encourage you to set up Timeshift. This tool creates system restore points.
    • Launch Timeshift. Select "RSYNC" as snapshot type (unless you have BTRFS specific setup).
    • Choose a snapshot location (preferably an external drive or a different internal partition, not your root system partition if possible).
    • Configure a schedule (e.g., daily, weekly).
  4. Driver Manager: Open "Driver Manager" from the menu. It will scan for proprietary hardware drivers (e.g., NVIDIA graphics, Broadcom Wi-Fi). Install recommended drivers if available. A reboot might be needed.
  5. Update Manager: Open "Update Manager" (shield icon in the panel).
    • You might be asked about an update policy; the default is usually fine.
    • Click "Refresh" to check for updates, then "Install Updates" to apply them. This updates your system and installed software.
  6. System Settings: Explore "System Settings" to customize your desktop, power management, display, etc.
  7. Software Manager: Use "Software Manager" to easily find and install thousands of applications. You can also use Synaptic Package Manager or the terminal (sudo apt install <packagename>).
  8. Firewall (GUFW): Linux Mint comes with ufw (Uncomplicated Firewall). You can manage it graphically with GUFW. sudo apt install gufw (if not pre-installed)
    Launch "Firewall Configuration", set Status to ON. Default "Incoming: Deny, Outgoing: Allow" is good for desktops.

Troubleshooting Tips

  • System Doesn't Boot / GRUB Issues: Check BIOS/UEFI boot order. Secure Boot (try disabling if issues). Use a live USB and the "Boot Repair" tool.
  • No Wi-Fi / Sound / Graphics: Use the "Driver Manager". Check Mint forums for your specific hardware model.
